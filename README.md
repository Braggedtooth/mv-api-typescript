# Mv-api-nestjs

## Description

Rewriting an express app using nestjs.
[Original](https://github.com/Braggedtooth/MV-Backend) 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
